Codelet

Codelet is a flashcard generation app that also serves as a helpful study tool. Codelet was built with a React frontend and a Flask backend. Multiple 3rd party libraraies were used to acheive a plethora of features.

Codelet is designed with software developers in mind. The seed data for the site includes over 20+ topics that any programmer can use to aid their studies. Within each flashcard set, a user can study with a flash card flip, play a matching game, quiz themselves on the topic, and listen to the term/definition with text-to-speech. MaterialUI was leveraged to implement highly reactive components in a timely manner.

Codelet's users were authenticated using Auth0. Any user can access the site and study, with a few limitations. If a user is logged in they can create new sets, favorite other sets, and also up/down-vote on other sets.

Codelet's frontend was hosted on AWS Amplify, adn the backend was hosted on Heroku. The live site can be found here.
