DisChat

DisChat was engineered using an express Frontend with vanillaJS, and an express backend utilizing Sequelize. The live-chat experience was optimized using websockets to update channels, users, and messages in real time, and AJAX was used to fetch this same info between page clicks.

The intersting part about DisChat is it resembles the functionality of a Frontend framework, such as React, in the sense that the main application page never reloads. This was tough to do with just AJAX, but the functionality made it well worth it.

Users were authenticated in DisChat using Bcrypt to hash passwords before being sent to the database. This provided ultimate security for every user's account information.

DisChat was hosted on Heroku and can be found here.
